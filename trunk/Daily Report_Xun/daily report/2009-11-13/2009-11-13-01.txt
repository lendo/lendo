1.用花括号括起来的若干条Java语句称为块。
   块确定了变量的作用域。
   一个块可以嵌套在另一个块中。
   但不能在嵌套的两个块中声明同命的变量。
   
2.if条件语句的格式为：if (condition) statement。
  条件为真时如果要执行多条语句，用块语句包含起来。如：
  {
      statement1;
      statement2;
      ...
  }
  
3.另一种条件语句为：if (condition) {statement1} else {statement2}；
  else子句与最邻近的if构成一组，清晰表示如：
  if (condition) 
  {
      if (condition)
          { 
              statement1;
          }
      else 
          {
              statement2；
          }
  }
  常见的情况也会出现如：if...else if...
  
4.while循环语句格式为：while (condition) statement 。
  条件为true时，执行while语句。
  条件为false时，不执行while语句。
  如果需要循环至少执行一次，可以使用 do/while语句。格式为：
  do statement while (condition)；
  do/while语句执行顺序为：先执行语句，检测循环条件，重复执行语句，再检测循环条件，以此循环执行。
  
5.for循环语句，格式为for( condition1; condition2; condition3)
  for语句的第1部分为初始化，
  第2部分为循环执行前检测循环条件，
  第3部分为更新初始化内容。
  for语句的一条不成文规则为：
  for语句的3个部分应该对同一个计数器变量进行初始化、检测和更新.
  在循环中，最好不要检测两个浮点数是否相等。比如0.1无法用二进制精确表示。
  for语句的第一部分声明变量之后，作用域为此for语句的循环体中。
  如果要在循环体外部使用计数器，使作用域更大的话。就要让这个变量在循环
  语句前面并且声明它。
  在不同循环体中定义的变量可以取同名。
  
6.带标签的break语句，在循环体外，前面声明时，执行到带标签的break
  语句时，就跳到标签处。
  标签声明后面紧跟冒号。
  
7.需要使用大数值时，可以用java.math包中 BigInteger类（整数）和
  BigDecimal类（浮点数）。
  处理大数值的运算时，要调用 java.math.BigInteger.方法 和
  java.math.BigDecimal.方法 。
  
8.数组是一种数据结构，用来存储同一数据类型值的集合。
  声明并初始化如：int[] a = new int[100]; ：声明数组a，并可以存放100个整数。
  还有种数组初始化方法如：int[] a = {1,2,3,4,5};即建立数组a，包含5个元素，值分别为1，2，3，4，5。
  还可以初始化匿名数组如：new int[]{2,3,4,5,6};。这种方式一般用于不创建新变量时初始化另一个数组。
  数组建好后就可以赋值了，一般用循环for语句赋值每个元素。
  计算元素个数，可以调用 array.length 方法。
  数组建好后，就不能改变建好的元素个数。如果要改，要用另一种数据结构：数组列表（array list）。
  
9.For each循环格式为：for(variable : collection) statement 。
  定义了一个变量，用来表示集合中的每一个元素，并执行后面语句。
  for each循环语句将处理每一个元素，不需要考虑数组的下标。
  
10.数组变量允许被拷贝给另一个数组变量相应的元素值。
   如果要拷贝一个数组的所有值，就要用Array类的copyOf方法。
   copyOf方法可以拷贝数组全部元素值给另一个数组，
   也可以拷贝了所有元素后增加新数组元素个数，如果是数值型多出来的元素赋值0，布尔型的赋值为false。

11.Java应用程序都有一个main方法带String arg[]参数。
   这个数组表示将接收命令行参数。
   
12.Arrays类中的sort方法用来对数值型数组进行从小至大排序。

13.如果要表示表格或更复杂的排列形式时，就要使用多维数组。
   声明初始化方式如：double[][] a = new double[i][j]; 。
   需要调用new来初始化数组。
   Java多维数组的另一种解释为：
   只有一维数组，多维数组就是"数组的数组"。
   如：a[i][j]: a数组包含i个数组元素，每个数组元素又包含j个元素的数组。
