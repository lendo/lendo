1.ORDER BY 语句用于对结果集进行排序。
  ORDER BY 语句默认按照升序对记录进行排序。
  如果希望对记录进行降序排列，可以使用关键字 DESC 。
  如果一列是字母组合，则以首字母顺序排列。
  例如：
     SELECT Company FROM Orders ORDER BY Company
     SELECT Company FROM Orders ORDER BY Company DESC
     SELECT Company,OrderNumber FROM Orders ORDER BY Company ASC,OrderNumber DESC
     
2.INSERT INTO 语句用于向表格中插入新的行。
  语法：
  INSERT INTO 表名称 VALUES (值1，值2，...)
  也可以指定插入的列：
  INSERT INTO 表名称 （列1，列2，...） VALUES (值1，值2，...)
  例如：
  INSERT INTO Persons VALUES ('Gates','Bill','Xuanwumen11','Beijing')
  INSERT INTO Persons ('LastName','Address') VALUES ('William','Chengdu')
  
3.SQL语言是一种声明式计算机语言
  不像C、Cobol、Java、汇编语言、Fortran、Visual Basic等
  过程式的计算机语言。
  SQL语言针对的是计算结果，
  过程式的计算机语言针对的是描述结果的过程。
